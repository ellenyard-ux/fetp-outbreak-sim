# Hospital Assets

This folder contains images for the District Hospital visual layout.

## Required Images

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `hospital_exterior.png` | Hero image of hospital exterior | 1200x600px |
| `hospital_ward.png` | Hospital ward/patient area | 800x600px |
| `hospital_ward_thumb.png` | Thumbnail for ward card | 400x300px |
| `hospital_lab.png` | Laboratory interior | 800x600px |
| `hospital_lab_thumb.png` | Thumbnail for lab card | 400x300px |
| `hospital_admin.png` | Administration office | 800x600px |
| `hospital_admin_thumb.png` | Thumbnail for admin card | 400x300px |

## Notes

- Images should be in PNG or JPG format
- The visual layout will show placeholder icons if images are missing
- Thumbnail images are optional - the main image will be used if no thumbnail exists
